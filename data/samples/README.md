# data/samples

В этой папке должны лежать **тестовые изображения** для проверки бота
и для скриншотов в курсовой работе.

## Что сюда положить

Подойдут любые картинки с печатным текстом на русском или английском:

- скриншот страницы из учебника или PDF;
- фото обложки книги с крупным заголовком;
- скриншот фрагмента документации;
- сгенерированная картинка с текстом (например, через `cv2.putText`
  или Pillow).

## Что сюда **не** класть

- личные документы, паспорта, банковские карты, медицинские справки;
- скриншоты переписки с реальными людьми без их согласия;
- любые материалы, которые нельзя выкладывать в публичный репозиторий.

Файлы из этой папки попадают в git, поэтому относитесь к ним
как к публичным.

## Идеи для демонстрации преподавателю

1. **Чистый печатный текст** (книга, статья) — показывает базовое качество.
2. **Скриншот с экрана** — типичный кейс «отправь скриншот ошибки».
3. **Фото с лёгким наклоном/тенью** — показывает работу предобработки.
4. **Картинка только с английским текстом** — демонстрация мультиязычности.
5. **Не-изображение** (например, .txt, отправленный как документ) —
   проверка обработки ошибок.
