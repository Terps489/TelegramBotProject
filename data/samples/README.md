# data/samples

Папка для тестовых изображений с печатным текстом (русский/английский).

Сюда можно положить:
- скриншоты страниц из учебника или PDF;
- фото обложек книг с крупным заголовком;
- скриншоты фрагментов документации;
- сгенерированные картинки с текстом.

Не клади сюда:
- личные документы, паспорта, банковские карты, медицинские справки;
- скриншоты переписки без согласия участников;
- любые материалы, которые нельзя выкладывать в публичный репозиторий.
